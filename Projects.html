Scratch Projects

I have created a few projects and scratch that have both educational and recreational capabilities. They are openly released on the repl.it websites and scratch websites, which host the capabilities of these programs. Most of these were assigned to be created in our AP Computer Science Principles class.

    Python Tips and Tricks

This program was created in scratch and gives tips and examples of python programs and syntax. This is an effort to teach people about programming in python, however this also taught me how to create and use variables, how to make functions, and how to use broadcasting.

    Scratch Galaga

This program was also created in scratch is a simple program that replicates the classic arcade game Galaga, where you play as a space ship shooting down aliens. This very recreational program has taught me how to use scratch for sprite movement, collision detection, and animations.

Repl.it Projects

I have also made a few repl.it projects which demonstrate the capabilities of python.

    Lunch program

This python program demonstrates potential factors that decide whether or not someone would get lunch. This showed me how to create functions, call those functions, create inputs, assign variables, and utilize if, then statements.

    Python Text Adventure

This next program is a compelling choose your own adventure story that gives many unique and surprising endings. By creating this program, it demonstrates the same ideas and practices as the lunch program, just in a much larger and complex scale, as well as demonstrating text structure, and text color changing.
